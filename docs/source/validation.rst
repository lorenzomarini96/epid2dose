Validation
==========

If reference Portal Dose distributions are available, **epid2dose**
automatically performs gamma-index analysis and generates a PDF validation
report.

.. image:: images/prediction_example.png
   :width: 850px
   :align: center

The generated validation report includes

* Reference Portal Dose
* Predicted Portal Dose
* Gamma-index histogram
* Gamma passing rate
