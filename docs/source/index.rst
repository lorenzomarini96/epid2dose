Welcome to epid2dose's documentation!
=====================================

**epid2dose** is an open-source deep learning framework for predicting
two-dimensional Portal Dose distributions directly from transit EPID images.

The package provides:

- automatic EPID preprocessing;
- Portal Dose prediction using an ensemble of U-Net models;
- gamma-index validation;
- automatic PDF report generation.
